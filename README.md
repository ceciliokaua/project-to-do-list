# 🚀 To-Do List

Uma aplicação de gerenciamento de tarefas, focada em produtividade e experiência do usuário, construída com tecnologias web puras.

## ✨ Funcionalidades
- **Gestão Completa:** Adicione, conclua, filtre e restaure tarefas.
- **Data e Hora:** Organize seus compromissos com precisão cronológica.
- **Lixeira Inteligente:** Visualize itens excluídos na aba "Excluídas" e restaure-os se necessário.
- **Tema Dinâmico:** Alternância entre Modo Claro e Modo Escuro com persistência de preferência.
- **Interface Glassmorphism:** Design futurista com efeitos de transparência e desfoque.

## 💾 Armazenamento e Privacidade
> **Nota Importante:** Este projeto utiliza o **LocalStorage** do navegador para salvar as informações. 
> 
> Isso significa que:
> - Seus dados ficam salvos **apenas localmente** no seu dispositivo.
> - As tarefas não são compartilhadas entre diferentes navegadores ou computadores.
> - Se você limpar os dados de navegação (cache) do seu browser, as tarefas serão removidas.
> - Não é necessário criar conta ou fazer login para utilizar.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Variáveis customizadas, Flexbox, Grid e animações @keyframes.
- **JavaScript (ES6+):** Manipulação de DOM, filtragem de Arrays e persistência de dados (JSON).
